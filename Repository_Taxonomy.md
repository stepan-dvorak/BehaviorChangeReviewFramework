# Repository Taxonomy

## Purpose
Defines the canonical classification of all repository content.

## Categories
### Governance
Repository rules and editorial guidance.
Examples:
- README.md
- CONTRIBUTING_FOR_AI.md
- AGENTS.md
- EDITORIAL_STYLE_GUIDE.md

### Core Research
Long-lived documents defining the framework.
- 00_Project_Charter.md
- 01_Terminology.md
- 02_Research_Methodology.md
- 03_Related_Work.md

### Standards
ISO, SEI and other external reference analyses.

### Microsoft Studies
Analyses of Microsoft guidance and platform architecture.

### Empirical Studies
Evidence collected from Business Central applications.

### Framework
Mechanism-independent framework documents.

### Archive
Superseded or historical material.

## Lifecycle
Draft → Review → Stable → Archived
